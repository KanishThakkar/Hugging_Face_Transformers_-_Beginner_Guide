##  What is Hugging Face Transformers?

Hugging Face Transformers is a powerful Python library that provides:

- Pre-trained NLP models  
- Easy-to-use pipelines  
- Support for tasks like:
  - Sentiment Analysis  
  - Text Summarization  
  - Question Answering  
  - Named Entity Recognition  
  - Text Generation  

It allows beginners to use advanced AI models with just a few lines of code.
